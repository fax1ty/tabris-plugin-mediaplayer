# Tabris.JS MediaPlayer Plugin

## Example

```ts
let audioPlayer = new AudioPlayer({ url: 'urlHere' });
audioPlayer.play();
```

or

```ts
new AudioPlayer({ url: 'urlhere', autoPlay: true });
```

To stop player from playing

```ts
audioPlayer.dispose();
```
